# あと何駅 プロトタイプ

京王線と山手線に絞った、電車内向けの現在地推定Webアプリです。

## 使い方

1. 公開URLをiPhoneのSafariで開く
2. 路線、方面、降車駅を選ぶ
3. `GPSで開始` を押す
4. GPSが取れない時やズレた時は、現在駅を選んで `この駅に補正` を押す

## iPhoneで試す

GitHub PagesでHTTPS公開して、外出先のiPhoneから使う前提です。

ホーム画面に追加する場合は、Safariの共有メニューから `ホーム画面に追加` を選びます。

位置情報をiPhoneで使うにはHTTPSが必要です。

## 初期仕様

- 対象路線: 京王線、山手線
- 表示: あと何駅、現在推定、次の駅、降車駅、簡易路線図
- 推定: GPSで最寄り駅に同期し、以後は駅間の概算時間で進行
- PWA: ホーム画面追加用メタ情報、Web App Manifest、Service Worker
- 通知、振動、遅延情報、種別判定は未実装

## 注意

駅座標と駅間時間はプロトタイプ用の概算です。
地下、遅延、快速や特急の通過駅にはまだ対応していません。
